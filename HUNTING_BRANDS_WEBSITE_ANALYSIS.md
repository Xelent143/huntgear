## 📐 Common Layout Structure (All Brands)

### Header/Navigation Pattern

```
┌────────────────────────────────────────────────────────────────────┐
│ [LOGO]    SHOP ▼ | HUNT ▼ | ABOUT | TECHNOLOGY        [SEARCH] [CART] │
│                                                                     │
│ Sub-nav: Jackets | Pants | Base Layers | Accessories | New Arrivals │
└────────────────────────────────────────────────────────────────────┘
```

**Key Elements:**
1. **Sticky Navigation** - Fixed on scroll (all brands)
2. **Mega Menu** - Large dropdowns with category images
3. **Hunt/Pursuit Dropdown** - Organized by animal (Deer, Elk, Waterfowl, etc.)
4. **Search Icon** - Prominent, often with predictive search
5. **Cart** - Always visible with item count

### Font Pairings Used by Top Brands

| Brand | Heading Font | Body Font | Style |
|-------|-------------|-----------|-------|
| Sitka | Custom Condensed | Helvetica Neue | Technical |

| First Lite | Serif (Playfair) | Sans-serif | Heritage |

### Recommended for Xelent Huntgear

```css
/* HEADINGS - Condensed, bold, uppercase */
font-family: 'Oswald', 'Roboto Condensed', sans-serif;
font-weight: 700;
letter-spacing: 0.02em;
text-transform: uppercase;

/* BODY - Clean, readable */
font-family: 'Inter', 'Helvetica Neue', sans-serif;
font-weight: 400;
line-height: 1.6;

/* ACCENT - Italic, lighter */
font-style: italic;
font-weight: 300;
```

## 🖼️ Image Guidelines

### Hero Images
- **Resolution**: 1920x1080 minimum
- **Treatment**: Dark overlay (50-70% black)
- **Subject**: Hunter in action, scenic wilderness, gear detail shots
- **Style**: Cinematic, dramatic lighting, early morning/late afternoon

### Product Images
- **Background**: Transparent or lifestyle context
- **Angles**: Front, back, detail shots
- **Lifestyle**: Model wearing gear in hunting environment
- **Consistency**: Same lighting/setup across all products

### Category Images
- **Aspect Ratio**: 4:5 (portrait) or 1:1 (square)
- **Treatment**: Dark gradient from bottom for text overlay
- **Subject**: Representative product or scene

## ⚡ Animation & Interaction Patterns

### Common Animations
| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| Hero Text | Fade up + stagger | 0.8s | cubic-bezier(0.16, 1, 0.3, 1) |
| Cards | Scale on hover | 0.3s | ease-out |
| Buttons | Color shift | 0.2s | ease |
| Images | Zoom on hover | 0.5s | ease-out |
| Scroll | Fade in | 0.6s | ease-out |
| Navigation | Slide down | 0.3s | ease |

### Scroll-Triggered Effects
- **Stats**: Count up animation when in viewport
- **Images**: Parallax subtle movement
- **Text**: Fade in from bottom
- **Cards**: Staggered reveal
